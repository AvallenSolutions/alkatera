# AlkaTera Landing Page

The main landing page for the AlkaTera platform.

## Features
- **Hero**: High-impact nature imagery with bold typography.
- **Manifesto**: Philosophy section with interactive hover effects.
- **Process**: Interactive tabs showing Analyse, Calculate, Strategise workflow.
- **Pricing**: 3-tier pricing cards (Seed, Growth, Canopy).
- **Contact**: Modal form for lead generation.
