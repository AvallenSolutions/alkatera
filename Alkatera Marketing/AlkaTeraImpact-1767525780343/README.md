# AlkaTera Impact Page

A visually stunning "Impact" page featuring generative art driven by sustainability metrics ("Living Data" direction).

## Features

- **Generative Art Visualization**:
    - **Carbon**: Recursive fractal trees that grow and sway (SVG-based).
    - **Water**: Animated ripple effects simulating rain or water impact.
    - **Waste**: Floating particles representing purification and reduction.
- **Interactive Metrics**: Hovering over specific metrics (Carbon, Water, Waste) isolates the corresponding visual layer, creating a "focus mode" for the user.
- **Organic Layout**: Clean, dark aesthetic (`#050505`) with neon accents (`#ccff00`, `#00ccff`) that pop against the background.
- **Case Studies**: Elegant editorial-style cards showcasing real-world impact.

## Dependencies

- framer-motion
- lucide-react
- clsx
- tailwind-merge
