# AlkaTera Manifesto Page

An immersive, emotionally resonant "Manifesto" page designed with the "Interactive Nature" aesthetic.

## Features

- **Organic Background**: A simulated "liquid" background using moving gradient blobs and blur filters to create an underwater/atmospheric feel.
- **Interactive Cursor**: A "light" follower that tracks the mouse, adding depth and interactivity to the dark background.
- **Scroll Animations**: Text elements blur in/out and slide up as you scroll, giving a dreamy, editorial reading experience.
- **Typography**: Giant serif headings paired with precise mono body text.
- **Sections**: Covers key themes of Water, Air (Carbon), and Harmony (Profit/Planet).

## Dependencies

- framer-motion
- lucide-react
- clsx
- tailwind-merge
