# AlkaTera Blog Post

A template for individual blog posts or detailed case studies. Designed for long-form content with a focus on readability and storytelling.

## Features

- **Article Header**: Immersive hero image with title and metadata.
- **Sidebar**: Sticky sidebar for key statistics, share buttons, and tags.
- **Rich Content Layout**: Two-column layout for Article text vs Sidebar.
- **Related Articles**: Suggestions for further reading.

## Dependencies

- framer-motion
- lucide-react
- clsx
- tailwind-merge

## Usage

```tsx
import { AlkaTeraBlogPost } from '@/sd-components/76fbe37b-184e-4994-991b-0e0b07388ae8';

function Page() {
  return <AlkaTeraBlogPost />;
}
```
